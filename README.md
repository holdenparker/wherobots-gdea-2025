# wherobots-gdea-2025
Wherobots Geospatial Data Engineering Associate Certification 2025

## Harlequin Wherobots
To run the Harlequin CLI IDE from the docker setup, run the following:
```sh
docker compose run harlequin
```

This will leverage the `gdea-cert.api` file, which contains the API key for Wherobots (and is git-ignored).
